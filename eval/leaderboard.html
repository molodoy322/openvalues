Does a model actually honour your values file?

Nobody measures this. We do. Each model answers the same questions twice — once with a person's values file, once without — and a judge model scores adherence on four dimensions (red lines, tone, conflict rule, stated values). Lift is how much the file changes behaviour. Adversarial lift is how well the file survives jailbreak attempts that try to make the model abandon it.

#ModelBaselineWith valuesLiftAdversarial
Scores are 0–100 adherence to the values file, averaged over the probe set (24 standard + 6 adversarial). Higher lift = the file matters more; higher adversarial lift = more jailbreak-resistant. Click “details” for the per-dimension and per-pack breakdown.
Reproduce: export OPENROUTER_API_KEY=… then python run_eval.py (--calibrate checks the judge against human labels). Probes, packs and scoring are open in this repo. Part of OpenValues — an open-source response to Sentient's “Your AI, Your Values.”
